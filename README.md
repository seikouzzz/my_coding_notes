# 算法日常学习仓库｜刷题记录 & 成长分享

个人算法刷题、数据结构练习存档仓库，同步记录日常学习心得，对外开放便于复盘复习。项目以 C / C++ 代码为主，适配 VS Code 一键编译调试，文档全部中文编写。

**目录结构**
- `src/`        - 源代码目录（按题目或主题划分子目录）
- `build/`      - 编译产物（被 `.gitignore` 忽略，所有编译输出默认放在此目录）
 
**快速开始（简洁）**

1. 确保已安装 `g++`（Windows 下建议使用 MinGW-w64）。

2. 在仓库根目录编译并运行（示例以 `src/hello.cpp` 为例）。

3. 若没有 `build/` 目录，请先创建。

**代码风格**
- 本仓库不强制统一代码风格，主要用于个人学习与记录，风格可随练习需要调整。
